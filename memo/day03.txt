시멘틱(Semantic) 태그
	Samantic(의미론적인, 의미있는) 의미론적인 이란 뜻으로 시맨틱 태그는 
	의미있는 태그 라는 의미를 갖는다.

시맨틱 태그의 장점
	시각장애가 있는 사용자에게 텍스트 판독기(스크린 리더)를 통해 의미 있는
	내용을 전달하여 사용 가능하도록 제공(웹 표준, 웹 접근성)
	태그를 기반으로 페이지 내 검색 키워듸 우순순위 판단

시맨틱 태그 종류
	<header>	페이지의 머리말을 의미하는 헤더 영역
	<main>		사이트의 핵심 컨텐츠 표시하는 영역
			<nav> <aside> <section> <article> 등 포함 가능
	<nav>		일반적으로 메뉴(네비게이션 링크)를 포함하는 영역	
	<section>	주제별 문서의 콘텐츠를 표시하는 영역
			하나 이상의 <article> 포함 가능.
	<article>	섹션의 하위 항목으로 본문 이외의 개별 콘텐츠를 표시하는 영역
			<section> 안에 여러개의 <article> 포함 할 수 있다.
	<aside>		좌/우측 사이드 영역으로 일반적으로 사이바 라고 말한다.
	<footer>	일반적으로 화면의 제일 하단에 위치.
			회사소개/저작권/약관 등을 표시.
	그외		<mark> <figure> <summary> <address> <iframe>

	참고 : https://sangyeon96.gitbooks.io/do-it-html5-css3/content/Chapter10-2.html


레이아웃 관련 태그
	<div>		범위 구분(Division) 태그 - 블록으로 한줄을 모두 차지
	<span>		좁은 범위 구분(Span) 태그 - 인라인으로 영역만 차지

텍스트 관련 태그
	<h1> ~ <h6>	제목(Headline)을 나타내는 태그
	<p>		텍스트의 단락(Paragraph)을 만드는 태그
	<br>		텍스트의 줄바꿈<Break>을 만드는 태그	
	<strong><b>	텍스트를 굵게 강조(Strong)하기위한 태그
	<em><i>		기울인 텍스트를 입력해주는 태그

목록 태그
	웹 문서에 목록은 어떤 항목을 나열할 때 뿐 아니라 네비게이션의 목록 도는
	텍스트 콘첸츠 목록 등을 나열할 때 CCS와 함께 자주 이용된다.


	<ol>	순서있는(Ordered List) 목록을 나열한다.
	<ul>	순서없는(Unordered List) 목록을 나열한다.
	<li>	목록(List)를 표시하며 ul/ol의 하위 목록으로 작성한다.

OL 목록 태그 속성
	type 영문자나 로마 숫자 등으로 순서 속성 변경한다.
		type = "1"	숫자(기본값)
		type = "a" 	영문 소문자
		type = "A"	영문 대문자
		type = "i"	로마 숫자 소문자
		type = "I"	로마 숫자 대문자
	start 	1부터 시작되는 시작 번호를 임의의 숫자로 변경한다.
	
속성 사용 예
	type="1" start = "4" 숫자 4부터 시작된다.
	type="A" start = "4" 영문 D부터 시작된다.

설명 목록 태그
	설명목록(Description List)은 이름과 값(name & value) 형태로 된 목록을 말한다.

	<dl>	설명록록은 한쌍의 <dt><dl>태그를 갖는다.
	<dt>	태그와 값 부분을 지정한다.
	<dd>	태그로 구성된다.

사용예
	<dl>
		<dt>이름</dt>
		<dd>값</dd>
	</dl>



테이블 태그
	<table>		웹 문서에서 표 형식의 문서를 만들기 위한 태그
	<caption>	table caption로 테이블 요약 설명을 표시
	<tr>		table row의 약자로 표의 행 표시
	<td>		table data의 약자로 표의 열 표시(<tr> 태그 하위 태그)
	<tbody>		table body로 표에서 본문 영역을 나타내기 위한 태그로 사용
	<thead>		table head로 표에서 제목 줄을 가지고 있는 경우 제목 줄을 표시
	<th>		table header의 약자라 표의 헤더를 표시(<td> 대신 사용)

테이블 태그 사용예(단순 사용)
	<table>
		<tr>
			<td>섹션 1</td>
			<td>섹션 2</td>
		</tr>
		<tr>
			<td>섹션 1</td>
			<td>섹션 2</td>
		</tr>
	</table>

테이블 태그 사용예(thead, tbody 사용)
	<table>
		<thead>
			<tr>
				<th>학교</th>
				<th>창립년도</th>
			</tr>
		</thead>
		<tbody>
			<tr>
				<td>서울대학교</td>
				<td>1946</td>
			</tr>
			<tr>
				<td>고려대학교</td>
				<td>1905</td>
			</tr>
		</tbody>
	</table>
