git
	버전관리 시스템(VCS, Version Control System)

git 주요 기능
	버전관리 	git의 가장 기본적인 기능으로 문서의 버전(히스토리)를 관리한다.
	백업	로컬 저장소의 문서를 원격 저장소에 백업하는 기능을 제공한다.
	협업	원격 저장소에 백업된 문서를 푸시(push)업로드/ 풀(pull)다운로드하여 
		협업이 가능하다.

github(원격 저장소의 일종)
	git을 사용할 때 백업을 위해 원격 저장소(Hub)를 제공해 주는 서비스
	git을 사용시 원격 저장소로 가장 많이 사용하지만 다른 원격 저장소를
	구축하는 것도 가능하다.

git상태
	git 파일은 항상 다음 3가지 상태중 하나의 상태를 갖는다.
	modified -> staged -> committed
	
	modified		로컬에서 파일을 수정중인 상태0\
	staged		commit이 되기 전 상태 (임시 저장 상태)
	committed	수정사항들이 자신의 로컬 git 환경에 저장돼 있는 상태
